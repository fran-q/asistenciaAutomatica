
# Asistencia con Reconocimiento facial

Esta aplicacion tiene como objetivo administrar de forma segura y eficaz el registro de las asistencias de profesores a traves de reconocimiento facial, asegurando la integridad y confiabilidad de las asistencias del equipo estudiantil, aportando funcionalidades para el sector administrativo
